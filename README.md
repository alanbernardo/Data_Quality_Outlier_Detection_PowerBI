📌 Data Quality & Outlier Detection com Power BI

Este projeto tem como objetivo analisar a qualidade de dados de uma base de clientes, utilizando estatística descritiva e técnicas de detecção de outliers no Power BI.

🎯 Problema de Negócio

Bases de dados frequentemente apresentam:

valores inconsistentes
registros fora do padrão
erros que podem comprometer análises

Esses problemas impactam diretamente a tomada de decisão e a confiabilidade dos insights.

💡 Solução Proposta

Aplicação de técnicas de estatística descritiva para:

entender a distribuição dos dados
identificar padrões
detectar outliers
apoiar processos de limpeza e validação
📊 Dados Analisados

A base contém informações de clientes:

Idade
Peso
Altura
📈 Análises Realizadas
🔹 Estatística Descritiva

Foram calculadas as seguintes métricas:

Mínimo
Máximo
Média
Mediana
Desvio padrão

👉 Objetivo:
Compreender a distribuição e comportamento dos dados.

🔹 Detecção de Outliers

Foi realizada análise visual da variável altura, permitindo identificar valores fora do padrão esperado.

👉 Técnicas utilizadas:

análise de dispersão
comparação com média e mediana
identificação de extremos
🔹 Cálculo da Moda com DAX

Foi criada uma medida personalizada utilizando:

SUMMARIZE
COUNT
MAXX
FILTER
CONCATENATEX

👉 Demonstrando capacidade de construção de lógica analítica no Power BI.

📊 Resultados e Insights
A média e mediana próximas indicam distribuição relativamente estável
O desvio padrão evidencia a dispersão dos dados
Foram identificados valores potencialmente inconsistentes (outliers)
Dados fora do padrão podem comprometer análises e modelos futuros
📌 Recomendações
Implementar validação de dados na entrada
Tratar ou remover outliers antes da análise
Monitorar continuamente a qualidade dos dados
Criar regras de negócio para identificação automática de inconsistências
